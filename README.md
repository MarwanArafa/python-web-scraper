A robust web scraping tool built with **BeautifulSoup** and **Requests**.

## Features
* **Data Extraction:** Pulls product titles, prices, ratings, and availability status.
* **CSV Export:** Automatically formats data into a clean Excel-ready CSV file.
* **Error Handling:** Includes check for connection status and User-Agent headers to mimic a real browser.

## Usage
1. Install dependencies:
   `pip install -r requirements.txt`
2. Run the scraper:
   `python book_scraper.py`
3. Open `science_books_data.csv` to see results.
